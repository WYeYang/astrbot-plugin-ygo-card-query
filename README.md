# 自然语言查卡插件

## 功能介绍

这是一个为asrtbot设计的自然语言查卡插件，允许用户通过自然语言描述来查询卡片信息。

## 插件结构

```
astrbot-plugin/
├── card_query.py       # 插件核心实现
├── manifest.json       # 插件配置文件
└── README.md           # 插件说明文档
```

## 功能特性

- 支持通过自然语言查询卡片
- 提供 `query_card` 函数工具
- 支持按卡片名称、类型、属性等进行查询
- 支持简单的条件查询（如攻击力大于某个值）

## 使用示例

### 示例1：查询特定卡片

```
用户：帮我查找青眼白龙的信息
Bot：调用 query_card 函数
函数参数：{"query": "青眼白龙"}
```

### 示例2：按类型查询

```
用户：查找所有怪兽卡
Bot：调用 query_card 函数
函数参数：{"query": "怪兽卡"}
```

### 示例3：按属性查询

```
用户：搜索光属性的卡片
Bot：调用 query_card 函数
函数参数：{"query": "光属性"}
```

### 示例4：按攻击力查询

```
用户：查找攻击力大于2500的卡片
Bot：调用 query_card 函数
函数参数：{"query": "攻击力大于2500"}
```

## 实现说明

- 插件使用 `get_functions` 方法暴露 `query_card` 函数
- 函数接受一个 `query` 参数，为自然语言查询语句
- 内部使用简单的文本匹配逻辑处理查询
- 返回包含查询结果的JSON对象
- 目前使用模拟数据，实际应用中可以连接到真实的卡片数据库

## 安装方法

将插件目录复制到asrtbot的插件目录中，重启bot即可使用。
